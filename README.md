# customer-churn-prediction
# 📊 Customer Churn Prediction App

## 📌 Project Overview

This project predicts whether a customer is likely to churn (leave a service) using Machine Learning.
It also provides an interactive web interface built with Streamlit for real-time predictions.

---

## 🎯 Problem Statement

Customer churn is a major issue for businesses. Predicting churn helps companies take preventive actions to retain customers.

---

## 📊 Dataset

* IBM Telco Customer Churn Dataset
* Contains customer details like tenure, charges, services, and contract type

---

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit

---

## ⚙️ Features

* Data preprocessing and cleaning
* Label encoding of categorical variables
* Random Forest model for prediction
* Interactive UI using Streamlit
* Real-time churn prediction

---

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/customer-churn-prediction.git
```

2. Navigate to project folder:

```bash
cd customer-churn-prediction
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
